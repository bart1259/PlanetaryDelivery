# Planetary Delivery

A simple Unity game where you deliver packages between eight different locations on a small planet within a time limit. As you learn the map and improve your driving skills, you can complete more deliveries in less time. The project was completed in one week.

## Download

You can download the game from [Itch.io](https://bart1259.itch.io/planetarydelivery).
